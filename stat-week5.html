PART 1  ·  假設檢定基礎：用證據判斷一個主張

上週（Week 4）估計是「猜母體參數大約多少」。本週假設檢定（Hypothesis Testing）換個問法：先有一個主張（假設），再用樣本證據判斷「該不該推翻它」。核心精神像法庭——無罪推定，證據夠強才定罪。

虛無假設 H₀ vs 對立假設 HₐWeek 5-1~2
H₀ 虛無假設 Null

「維持現狀」的預設立場

  • 要被檢驗的既定信念，代表 Business as Usual。
  • 一定含「=」號（=、≤、≥）。
  • 檢定一開始先假設 H₀ 為真，除非證據推翻。
Hₐ（或 H₁）對立假設

「想證明的新主張」

H₀ 的相反面。研究者真正想成立的主張要放這裡；只有當樣本資料與 H₀ 矛盾時，才會轉而支持 Hₐ。

⚖️ 法庭類比（最好記）

H₀：被告無罪
innocent
vs
Hₐ：被告有罪
guilty
→
無罪推定
證據足才定罪
💡 三種應用情境：① 研究假設→想證明的放 Hₐ；② 檢驗廠商聲稱→聲稱先當 H₀（無罪推定），資料矛盾才推翻；③ 決策情境→如「收貨 vs 退貨」。只有「拒絕 H₀」時，資料分析者才能下確定的結論。
三種假設形式 + Metro EMS 範例Week 5-3
白話：對母體平均 μ 的檢定，一定是下面三種之一（μ₀ 是被檢驗的假設值）。H₀ 永遠含等號。左兩種是「單尾」，右一種是「雙尾」。
左尾（下單尾）
H₀: μ ≥ μ₀
Hₐ: μ < μ₀
右尾（上單尾）
H₀: μ ≤ μ₀
Hₐ: μ > μ₀
雙尾
H₀: μ = μ₀
Hₐ: μ ≠ μ₀

🔖 範例：Metro EMS（緊急醫療反應時間）

某市急救目標：平均反應時間 ≤ 12 分鐘。想用樣本檢驗「有沒有達標」。μ = 母體平均反應時間。

假設結論與行動
H₀: μ ≤ 12達標，無需後續行動（維持現狀）。
Hₐ: μ > 12未達標，需採取改善行動（研究者要證明的）。
📖 目標是「≤12」，所以維持現狀放 H₀；「超標＝服務不佳」是要警覺的事，放 Hₐ。這是「右尾檢定」。
型 I 與型 II 誤差Week 5-4~5
白話：用樣本判斷難免出錯，錯有兩種。記口訣：型 I ＝冤枉（好人被判有罪）；型 II ＝放縱（壞人沒被抓）。
型 I 誤差 α

冤枉：H₀ 為真卻被拒絕

α = P{拒絕 H₀ | H₀ 為真} = 生產者風險

α ＝顯著水準（level of significance），由檢定者事先指定的「最大可容忍冤枉機率」，常設 0.05。後果較嚴重。

型 II 誤差 β

放縱：H₀ 為假卻沒拒絕

β = P{不拒絕 H₀ | H₀ 為假} = 消費者風險

通常無法直接控制。統計學家因此說「不拒絕 H₀（do not reject）」而非「接受 H₀」——沒證據定罪 ≠ 證明無罪。

四格總表（勿枉勿縱！）

決策 ＼ 真相H₀ 為真H₀ 為假
不拒絕 H₀✓ 正確型 II 誤差 β（放縱）
拒絕 H₀型 I 誤差 α（冤枉）✓ 正確

檢定力 Power（公權力）

Power = 1 − β = P{拒絕 H₀ | H₀ 為假}

當 H₀ 確實為假時，「成功抓到」的機率。越高越好。

⚠️ 兩種錯不能同時壓低

β（放縱）會變大當：真值與假設值差距變小、α 變小、母體 σ 變大、樣本 n 變小。降低 α 常使 β 上升——像翹翹板，得靠「加大樣本／更多證據」才能兩邊都改善。

p 值與檢定步驟Week 5-6

p 值（p-value）是什麼

在「H₀ 為真」的前提下，觀察到目前這麼極端或更極端樣本結果的機率。p 值越小＝資料越不像 H₀ 為真時該有的樣子＝越該拒絕。

拒絕 H₀ ⟺ p 值 < α
💡 口訣：p 小則拒（small p → reject）；p 大則留（large p → not reject）。p 值就是「把檢定統計量換算成機率」，方便直接跟 α 比大小。

檢定七步驟

  1. 確立 H₀ 與 Hₐ。
  2. 選對的檢定統計量（Z / t / χ² / F）。
  3. 訂顯著水準 α。
  4. 用 α 定「拒絕規則」（臨界值 / 拒絕區）。
  5. 收集樣本、算檢定統計量。
  6. (a) 比臨界值 或 (b) 算 p 值比 α。
  7. 下結論。

PART 2  ·  母體平均數 μ 檢定（σ 已知 或 n≥30，用 Z）

檢定統計量 Z＝(X̄−μ₀)/(σ/√n)：把「樣本平均離假設值多遠」換算成標準常態的距離。再看它有沒有落進拒絕區。分單尾、雙尾兩種。

單尾 Z 檢定 + Metro EMS 範例Week 5-7~8

檢定統計量與拒絕規則（以右尾 Hₐ: μ>μ₀ 為例）

Z = X̄ − μ₀σ/√n
規則1：Z > Zα 拒絕（右尾）
Z < −Zα 拒絕（左尾）
規則2：p 值 < α 拒絕
規則3：X̄ > μ₀+Zασ/√n 拒絕

🔖 範例：Metro EMS（右尾，α=0.05）

n=40，X̄=13.25 分，S=3.2 分（n≥30，S 可代 σ）。H₀: μ≤12，Hₐ: μ>12。

Z = (13.25 − 12) / (3.2/√40) = 1.25 / 0.506 = 2.47
拒絕規則：Z > Z0.05 = 1.645 → 2.47 > 1.645 ✅ 拒絕 H₀
p 值 = P(Z > 2.47) = 0.0068 < 0.05 → 拒絕 H₀
📖 結論：有 95% 信心 Metro EMS 未達 12 分鐘目標，應改善服務。檢定統計量 2.47 落進右尾拒絕區（>1.645），對應的尾端面積 p=0.0068 也 <0.05，兩種判法一致。
雙尾 Z 檢定 + Glow 牙膏範例Week 5-9~11

雙尾規則（Hₐ: μ ≠ μ₀）

拒絕 H₀ if |Z| > Zα/2（兩邊都算，臨界值用 α/2）
💡 信賴區間法（等價判法）：用樣本算 μ 的信賴區間；若假設值 μ₀ 落在區間內→不拒絕；落在區間外→拒絕。這與雙尾檢定結論永遠一致。

🔖 範例：Glow 牙膏（雙尾，α=0.05）

生產線目標每支重 6 盎司。抽 n=30，X̄=6.1，S=0.2。偏離 6 就要停機調整。H₀: μ=6，Hₐ: μ≠6。

Z = (6.1 − 6) / (0.2/√30) = 0.1 / 0.0365 = 2.74
|Z|=2.74 > Z0.025=1.96 → 拒絕 H₀；p 值 = 2×P(Z>2.74) = 2(0.0031) = 0.0062 < 0.05
信賴區間法：6.1 ± 1.96(0.2)/√30 = [6.0284, 6.1716]，不含 μ₀=6 → 拒絕
📖 結論：有 95% 信心平均填充重量不是 6 盎司，應停機調整。三種判法（臨界值、p 值、信賴區間）結論完全一致——這是雙尾檢定與區間估計互通的漂亮之處。

PART 3  ·  母體平均數 μ 檢定（σ 未知，n<30，用 t）

σ 未知又是小樣本，就把 Z 換成 t，臨界值改查 t 表（自由度 n−1）。做法完全一樣，只是分配尾巴更胖、更保守。最後補上「怎麼算型 II 誤差 β 與檢定力」。

t 檢定規則Week 5-12

檢定統計量

t = X̄ − μ₀S/√n  (自由度 df = n−1)
假設形式拒絕 H₀ 的條件
右尾 Hₐ: μ>μ₀t > tα,n−1
左尾 Hₐ: μ<μ₀t < −tα,n−1
雙尾 Hₐ: μ≠μ₀|t| > tα/2,n−1
⚠️ 一般 t 表精度不足以給出「精確 p 值」，只能查出 p 值的範圍（如「p < 0.005」）。要精確 p 值得靠電腦軟體。
🔖 範例：Highway Patrol 測速（右尾 t）Week 5-13~14

州警檢定某路段車速，H₀: μ≤65。拒絕 H₀ 的地點適合設測速照相。地點 F：n=16，X̄=68.2 mph，S=3.8，α=0.05。

t = (68.2 − 65) / (3.8/√16) = 3.2 / 0.95 = 3.37，df = 15
拒絕規則：t > t0.05,15 = 1.753 → 3.37 > 1.753 ✅ 拒絕 H₀；p 值 < 0.005
信賴區間法：μ < 68.2 − 1.753(3.8)/√16 = 66.535，而 μ₀=65 < 66.535 → 拒絕
📖 結論：有 95% 信心地點 F 車速平均 > 65 mph，是設測速照相的好地點。X̄=68.2 落在臨界值 66.665 右邊的拒絕區。
計算型 II 誤差 β 與檢定力Week 5-14~15
白話：前面都在控制型 I 誤差 α。那「放縱」的機率 β 怎麼算？步驟：① 先由 α 定出臨界值 a；② 假設「真值其實是某個 μ'」，算「X̄ 仍落在不拒絕區」的機率＝β。

🔖 範例：n=100, σ=8.9, μ₀=70, α=0.05（右尾）

先做檢定：X̄=71.8 → Z=(71.8−70)/(8.9/√100)=1.8/0.89=2.02 > 1.645 → 拒絕 H₀
臨界值 a = μ₀ + Zα·σ/√n = 70 + 1.645(0.89) = 71.464（X̄ 超過它就拒絕）

若真值 μ=72，β = ?

β = P(X̄ ≤ 71.464 | μ=72)
= P(Z ≤ (71.464−72)/0.89)
= P(Z ≤ −0.6) = 0.2743

若真平均是 72，有 27.43% 機率仍誤判成「μ=70」（放縱）。

檢定力 Power = ?

Power = 1 − β
= 1 − 0.2743 = 0.7257

若真平均是 72，有 72.57% 機率能正確抓到「μ≠70」。

特徵曲線 OC Curve（讀圖方法）

📖 讀圖：橫軸 δ=|μ−μ₀| 是「真值離假設值多遠」。曲線由左往右下降：真值離 μ₀ 越遠（δ 越大），越容易被抓到→β（放縱率）越小。δ 很小時兩個分配幾乎重疊，很難分辨，β 就大。

PART 4  ·  母體比率 p 檢定

檢定「支持率、不良率、比例」是否等於某個聲稱值 p₀。統計量把樣本比率 p̂ 標準化，標準誤用假設值 p₀（不是 p̂）算。條件：np₀≥5 且 n(1−p₀)≥5。

比率 p 檢定 + NSC 酒駕範例Week 5-16~17

檢定統計量

Z = p̂ − p₀√[p₀(1−p₀)/n]
💡 注意：分母用假設值 p₀ 算標準誤（因為檢定是「假設 H₀ 為真」時看資料有多極端），這跟上週「區間估計用 p̂」不同。拒絕規則：右尾 Z>Zα、左尾 Z<−Zα、雙尾 |Z|>Zα/2。

🔖 範例：NSC（酒駕比例，雙尾 α=0.05）

NSC 聲稱 50% 車禍肇因於酒駕。抽 120 件車禍，67 件酒駕。H₀: p=0.5，Hₐ: p≠0.5。

p̂ = 67/120 = 0.5583
Z = (0.5583 − 0.5) / √[0.5(0.5)/120] = 0.0583 / 0.0456 = 1.278
|Z|=1.278 < Z0.025=1.96 → 不拒絕 H₀；p 值 > 0.1 > 0.05
📖 結論：證據不足以推翻「50%」的聲稱。注意這不是「證明剛好 50%」，只是「沒有足夠證據說不是 50%」（不拒絕 ≠ 接受）。

PART 5  ·  母體變異數 σ² 檢定（卡方）

檢定「變異／穩定度」是否等於某值。品管常見：波動是否過大？統計量用卡方，因為卡方右偏、非對稱，雙尾的左右臨界值不一樣。前提：母體常態。

卡方 σ² 檢定 + 房租範例Week 5-18~20

檢定統計量

χ² = (n−1)S²σ₀² ≈ χ²n−1
假設形式拒絕 H₀ 的條件
右尾 Hₐ: σ²>σ₀²χ² > χ²α,n−1
左尾 Hₐ: σ²<σ₀²χ² < χ²1−α,n−1
雙尾 Hₐ: σ²≠σ₀²χ² < χ²1−α/2,n−1 或 χ² > χ²α/2,n−1

🔖 範例：Apartment Rents（雙尾 α=0.05）

n=10，S=$60。檢定母體標準差是否為 65。H₀: σ²=65²，Hₐ: σ²≠65²。df=9，χ²0.025,9=19.02、χ²0.975,9=2.70。

χ² = (10−1)(60²) / (65²) = 32400 / 4225 = 7.67
拒絕區：χ²<2.70 或 χ²>19.02。而 2.70 < 7.67 < 19.02 → 不拒絕 H₀（p 值 > 0.1）
📖 結論：χ²=7.67 落在兩臨界值之間（不拒絕區），沒有足夠證據說標準差不是 65。等價地，σ 的信賴區間 [41.3, 109.5] 含 65，也不拒絕。

PART 6  ·  兩母體差異的檢定

檢定「兩組有沒有差」：兩母體平均差 μ₁−μ₂、兩母體比率差 p₁−p₂。H₀ 通常是「差＝0」（沒差別）。分獨立樣本 / 配對樣本兩種設計。

兩母體平均差（σ 已知，用 Z）Week 5-21

檢定統計量（H₀: μ₁−μ₂=0）

σ₁≠σ₂：

Z = (X̄₁−X̄₂) − 0√(σ₁²/n₁+σ₂²/n₂)

σ₁=σ₂=σ：

Z = (X̄₁−X̄₂) − 0σ√(1/n₁+1/n₂)

拒絕規則：右尾 Z>Zα、左尾 Z<−Zα、雙尾 |Z|>Zα/2。

兩母體平均差（σ 未知，用 t）Week 5-22
σ₁ ≠ σ₂

各用各的 S²

t = (X̄₁−X̄₂)−0√(S₁²/n₁+S₂²/n₂)

自由度 v 用 Welch 公式（計算機自動處理）。

σ₁ = σ₂

合併變異數 Sp²

Sp² = (n₁−1)S₁²+(n₂−1)S₂²n₁+n₂−2
t = (X̄₁−X̄₂)−0Sp√(1/n₁+1/n₂)

v = n₁+n₂−2。

💡 大／小樣本：n₁,n₂≥30 → t≈Z（可用 Z 近似）；n₁,n₂<30 → 務必用 t 分配。
🔖 範例：Par 高爾夫球（右尾，α=0.01）Week 5-23

Par 球平均飛行距離是否 > Rap 球？n₁=120,X̄₁=235,S₁=15；n₂=80,X̄₂=218,S₂=20。H₀: μ₁−μ₂≤0，Hₐ: μ₁−μ₂>0。

Z = (235−218) − 0 / √(15²/120 + 20²/80) = 17 / 2.62 = 6.49
拒絕規則：Z > Z0.01 = 2.33 → 6.49 > 2.33 ✅ 拒絕 H₀
📖 結論：至少 99% 信心，Par 球平均飛行距離大於 Rap 球。（大樣本用 Z 近似 t。）
配對樣本 + Express Deliveries 範例Week 5-24~26
白話：配對（matched）＝同一對象測兩次或成對比較。做法：先算每對的差 d，再對「差」做單母體 t 檢定（檢定 μ_d 是否為 0）。

檢定統計量

t = d̄ − μ_dS_d/√n  (μ_d 通常設 0，df = n−1)

🔖 範例：兩家快遞 UPX vs INTEX（雙尾 α=0.05）

同 10 個分公司各寄兩份文件（一家一份），比較送達時間差。兩服務送達時間有無差異？n=10，d̄=2.7，S_d=2.9。H₀: μ_d=0。

t = (2.7 − 0) / (2.9/√10) = 2.7 / 0.917 = 2.94，df=9
|t|=2.94 > t0.025,9=2.262 → 拒絕 H₀
📖 結論：t=2.94 落進右尾拒絕區，兩家快遞的平均送達時間有顯著差異。配對設計消掉了「各分公司地理位置」造成的干擾，比獨立樣本更靈敏。
兩母體比率差 + MRA 廣告範例Week 5-27~28

檢定統計量（H₀: p₁−p₂=0，需先算合併比率 p̄）

p̄ = X₁+X₂n₁+n₂
Z = (p̂₁−p̂₂) − 0√[p̄(1−p̄)(1/n₁+1/n₂)]
💡 為何用合併比率 p̄？H₀ 說「兩比率相等（＝共同的 p）」，那就把兩組資料合起來估這個共同 p̄，用它算標準誤，最符合 H₀ 為真的假設。

🔖 範例：MRA 廣告成效（右尾 α=0.05）

活動後知名度 p₁ 是否 > 活動前 p₂？n₁=250,X₁=120,p̂₁=0.48；n₂=150,X₂=60,p̂₂=0.40。H₀: p₁−p₂≤0，Hₐ: p₁−p₂>0。

p̄ = (120+60)/(250+150) = 180/400 = 0.45
Z = (0.48−0.40) − 0 / √[0.45(0.55)(1/250+1/150)] = 0.08 / 0.0514 = 1.56
Z=1.56 < Z0.05=1.645 → 不拒絕 H₀（未顯著增加）
📖 結論：雖然樣本上活動後高了 8 個百分點，但證據不夠強（1.56 < 1.645），還不能斷定廣告確實提升了知名度。對照 Week 4 的區間 (−0.02, 0.18) 含 0，結論一致。

PART 7  ·  F 檢定：比較兩母體變異數

要判斷「兩組的變異／波動是否相同」——例如做兩母體平均差前要先決定 σ₁ 是否等於 σ₂。工具是兩樣本變異數的比值 F=S₁²/S₂²，服從 F 分配。

F 檢定 + Stocks 股利範例Week 5-29~30

檢定統計量與「倒數」陷阱

F = S₁²S₂²  (df₁=n₁−1, df₂=n₂−1)
⚠️ Note! Reverse（左臨界要倒數換算）：F 表通常只給右尾臨界值。雙尾的左臨界值要用倒數：F1−α/2; df₁,df₂ = 1 / Fα/2; df₂,df₁（注意自由度也對調！）。

🔖 範例：Stocks 股利波動（雙尾 α=0.05）

比較 NYSE 與 NASDAQ 股利殖利率的變異數是否不同。n₁=21,S₁=1.3；n₂=25,S₂=1.16。H₀: σ₁²=σ₂²。df₁=20, df₂=24。

F = 1.3² / 1.16² = 1.69 / 1.3456 = 1.256
右臨界 F0.025,20,24 = 2.33；左臨界 = 1 / F0.025,24,20 = 1/2.41 = 0.415
0.415 < F=1.256 < 2.33 → 不拒絕 H₀
📖 讀圖／結論：F=1.256 落在兩臨界值之間（不拒絕區），沒有證據顯示兩市場股利變異數不同。F 分配右偏，所以左右臨界值不對稱，左邊要用倒數換算。
