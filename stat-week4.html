PART 1  ·  估計基礎：從樣本猜母體

上週（Week 3）我們從「已知母體」往下推樣本會長怎樣（演繹法・機率論）。本週反過來：手上只有樣本，要往上猜「母體參數大概多少」——這就是估計（Estimation），屬於歸納法。先學怎麼判斷一個估計好不好（三大準則），再學兩種估計方式（點估計、區間估計）。

歸納法 Induction vs 演繹法 DeductionWeek 4-1
白話：統計學有兩個方向。演繹法（Deduction・機率論）＝從「已知的母體」往下算樣本會怎樣（上週在做的事）。歸納法（Induction・估計）＝反過來，只看得到樣本，要往上推「母體參數大約是多少」（本週在做的事）。一句話：根據事實或證據，去猜母體參數。

兩個方向的對照（讀圖方法：綠箭頭往上＝估計，白箭頭往下＝機率）

📖 讀圖：綠色箭頭（往上）＝從樣本統計量猜母體參數，這是「估計」；灰色箭頭（往下）＝從母體算樣本，這是上週的「機率論」。本週全部都在走綠色那條路。
估計準則① 無偏性 UnbiasedWeek 4-1
先講一個名詞：估計量（estimator）＝用來猜母體參數的公式，例如用樣本平均 X̄ 去猜母體平均 μ。「θ̂」（theta-hat，讀 theta-hat）是「拿來估計 θ 的統計量」的通用寫法，帽子＝估計值。

定義：無偏估計量（無偏估計）

如果一個估計量 θ̂ 的抽樣分配平均數剛好等於母體參數 θ，就叫它無偏估計量（unbiased estimator，無偏估計）。

E{θ̂} = θ
💡 白話：「無偏」不是說每次都猜對，而是「平均而言不偏不倚」——如果重複抽很多次樣本、每次都算一個 X̄，這些 X̄ 的中心會剛好落在真正的 μ 上，不會系統性偏高或偏低。

讀圖：無偏 vs 有偏（Biased）

📖 讀圖：紅色鐘形的正中央剛好對在真正的 μ 上→無偏；紫色鐘形的中央偏離了 μ→有偏（系統性地偏一邊）。我們偏好無偏的估計量。
估計準則② 有效性・③ 一致性Week 4-2

② 有效性 Efficiency（變異數最小）

在所有無偏估計量之中，變異數最小的那一個，稱為最有效估計量（most efficient estimator）。

💡 白話：大家都不偏（中心都對），那就比誰「抖動最小、最集中」。越集中，代表單次估計越可能靠近真值。例如估計中心時，樣本平均數 X̄ 就比樣本中位數更有效（更集中）。
📖 讀圖：兩條都對準 μ（都無偏），但青色（平均數）比較窄＝變異數小＝更有效。

③ 一致性 Consistency

當樣本數 n 越大，樣本平均數偏離母體平均數的機會就越小；n→∞ 時幾乎必然貼緊真值。

limn→∞ P(|X̄ − μ| ≥ ε) = 0  (任意 ε > 0)
📖 讀圖：樣本越大，鐘形越窄、越貼近 μ。「加大樣本就會越來越準」就是一致性。
點估計 vs 區間估計Week 4-2~3
點估計 Point Estimator

給一個「單一數字」

用一個統計量 θ̂ 的單一值去猜母體參數。例：用 X̄ = 75 直接猜 μ ≈ 75。
優點：簡潔。缺點：幾乎不可能剛好命中，也沒說「誤差多大」。

區間估計 Interval Estimator

給一段「範圍」

給一個區間把母體參數框住。
雙尾：θ̂L ≤ θ ≤ θ̂U
單尾：θ ≤ θ̂U 或 θ̂L ≤ θ
優點：同時表達「猜多少」與「有多不確定」。

💡 一句話抓重點：點估計像射一個「點」，區間估計像撒一張「網」。EMBA 更常用區間估計，因為它附帶了「信心水準」，能誠實說出估計的可信程度。
各分配的點估計公式Week 4-3
白話：不同的母體參數，用不同的樣本統計量去估。好消息是——最常見的幾個都直接用樣本平均數 X̄！

一般情形（估計 μ 與 σ²）

μ̂ = Σ Xin = X̄
σ̂² = Σ(Xi − X̄)²n − 1 = S²
⚠️ 注意一個陷阱：S²（樣本變異數）是 σ² 的無偏估計，但 S 本身不是 σ 的無偏估計！E(S) ≠ σ。若要無偏估計 σ，得除以修正係數 c₄：σ̂ = S / c₄，其中 c₄ = (2/(n−1))½ · Γ(n/2)/Γ((n−1)/2)。EMBA 實務上通常直接用 S 近似即可。

PART 2  ·  母體平均數 μ 的區間估計

最經典的問題：抽了一個樣本，算出 X̄，那母體真正的平均 μ 大約落在哪個範圍？答案分兩種情況——σ 已知（用 Z）與更常見的 σ 未知（用 t 分配）。再學怎麼反推「要抽多少人」才夠準。

區間估計的原理・信賴區間Week 4-4
從上週接上來：Week 3 學過 X̄ 的抽樣分配 → 中心 μ_X̄ = μ，標準誤 σ_X̄ = σ/√n。標準化後 Z=(X̄−μ)/(σ/√n) 服從標準常態。把「95% 的 Z 落在 ±Zα/2 之間」這句話的不等式重新整理，就得到 μ 的信賴區間。

從機率式推到信賴區間（三步驟）

① 標準常態：P{−Zα/2 ≤ Z ≤ Zα/2} = 1−α
② 代入 Z=(X̄−μ)/(σ/√n)，抽樣前（μ 已知）：P{μ − Zα/2·σ/√n ≤ X̄ ≤ μ + Zα/2·σ/√n} = 1−α
③ 抽樣後（μ 未知，反解 μ）→ 信賴區間 Confidence Interval：
X̄ − Zα/2·σ/√n ≤ μ ≤ X̄ + Zα/2·σ/√n
📖 讀圖：中間 1−α 的面積代表「有這麼高的信心」；兩尾各 α/2 是被排除的極端。右邊：以 X̄ 為中心，往兩邊各推 Zα/2·σ/√n，就框出信賴區間。
σ 已知：用 Z 的信賴區間 + 範例Week 4-5

(1−α) 信賴區間公式

X̄ ± Zα/2 · σ√n

(1−α) 稱為信賴係數 / 信賴水準（confidence coefficient / level）。常用：95%→Z=1.96；99%→Z=2.576；90%→Z=1.645。

💡 名詞：邊際誤差（margin of error, E）＝ Zα/2·σ/√n，就是「± 後面那一串」，代表點估計外推的半徑。

🔖 範例：National Discount（全國折扣連鎖）

某連鎖店評估新據點時，要估「該區居民平均年收入 μ」。抽 n=36 人，樣本平均 X̄=$21,100，標準差 σ=$4,500，用 95% 信賴水準。

邊際誤差 E = Zα/2·σ/√n = 1.96 × 4500/√36 = 1.96 × 750 = 1,470
區間估計 = 21,100 ± 1,470 → P{19,630 ≤ μ ≤ 22,570} = 0.95
📖 精確度陳述（Precision Statement）：「95% 的樣本平均會落在真值 μ 的 ±$1,470 內」。所以我們有 95% 信心，母體平均年收入介於 $19,630 到 $22,570。
σ 未知：改用 Student-t 分配Week 4-6~7
白話：現實中母體標準差 σ 幾乎都不知道，只能用樣本標準差 S 代替。但用 S 取代 σ 會多一份不確定，所以分配不再是常態，而是更胖尾的 t 分配。臨界值改用 tα/2, n−1。

(1−α) 信賴區間（σ 未知）

X̄ ± tα/2, n−1 · S√n
⚠️ n < 30：務必用 t 分配 tα/2, n−1。
💡 n ≥ 30：tα/2, n−1 ≈ Zα/2，用哪個都差不多。

t 分配的重點性質

  • 是一「家族」分配，由自由度 df（degrees of freedom，自由度）決定形狀，這裡 df = n−1。
  • 鐘形、對稱、平均數 = 0。
  • 比標準常態尾巴更胖（更保守，區間更寬）。
  • df 越大→越接近標準常態；df 越大→越集中。

讀圖：t vs Z（df 越小尾越胖）

📖 讀圖：紅色 Z 最高最瘦；df 越小的 t（青色）越矮、尾巴越翹（越胖）。所以小樣本用 t 得到的區間會比用 Z 更寬——這是誠實反映「樣本小、更不確定」。

🔖 範例：Apartment Rents（校區房租，t）Week 4-9

校刊記者調查校區半英里內套房月租。抽 n=10 間，X̄=$550，S=$60，母體假設常態，求 95% 信賴區間。

σ 未知 + n=10<30 → 用 t，df = n−1 = 9，t0.025,9 = 2.262

550 ± 42.92 → P{507.08 ≤ μ ≤ 592.92} = 0.95
📖 有 95% 信心，母體平均月租介於 $507.08 到 $592.92。（想自己換數字？用下方「μ 區間（σ 未知 t）」計算機。）
決定樣本數 n（平均數）Week 4-9~10
反過來問：如果我先規定「誤差不能超過 E」，那到底要抽幾個人？把邊際誤差公式 E = Zα/2·σ/√n 反解出 n 即可。

樣本數公式

n = (Zα/2)² · σ²E²
⚠️ n 算出來若有小數，一律無條件進位（要湊足精度，寧多勿少）。

🔖 範例：National Discount（要多準？）

管理層要求 95% 信心下，抽樣誤差 ≤ $500（σ=4,500）。要抽幾人？

n = (1.96)²(4500)² / (500)² = 311.17 → 進位 312
📖 越想要小誤差 E、或母體越分散 σ，需要的 n 就越大（n 與 E² 成反比、與 σ² 成正比）。

PART 3  ·  母體比率 p 的區間估計

從「平均數」換成「比率」——支持率、知名度、不良率都是比率。做法一樣：點估計 p̂ = X/n，再用常態近似加減邊際誤差。差別只在標準誤換成 √[p̂(1−p̂)/n]。

比率的信賴區間Week 4-10

點估計與抽樣分配

p̂ = Xn

當 n 夠大時，p̂ ≈ 常態，中心 p、標準誤 √[p(1−p)/n]。

(1−α) 信賴區間（n 大且 p̂ ≥ 0.1）

p̂ ± Zα/2 · √p̂(1−p̂)n
💡 初學者說明：p̂＝樣本比率（0~1）；n＝樣本數。真正的 p 未知，所以標準誤裡用 p̂ 代替 p。
🔖 範例：Political Science（選民民調）Week 4-11

民調公司電訪 500 位登記選民，有 220 位支持某候選人，求支持率 p 的 95% 信賴區間。

p̂ = 220/500 = 0.44
E = 1.96 × √[0.44(1−0.44)/500] = 1.96 × 0.0222 = 0.0435
0.44 ± 0.0435 → P{0.3965 ≤ p ≤ 0.4835} = 0.95
📖 有 95% 信心，真正的支持率介於 39.65% 到 48.35%。（媒體常說「誤差 ±4.35 個百分點」就是這個 E。）
決定樣本數 n（比率）Week 4-12

樣本數公式

n = (Zα/2)² · p(1−p)E²
⚠️ p 該用什麼？若有先前估計（如上例 0.44）就用它；若完全沒資訊，保守用 p=0.5——因為 p(1−p) 在 0.5 時最大，會給出「最大所需樣本數」，最安全。

🔖 範例：PSI（99% 信心，誤差 ≤ 0.03）

設定用的 pZ算式n
99% 信心0.44（先前）2.5762.576²·0.44·0.56 / 0.03²1817
99% 信心0.5（保守）2.5762.576²·0.25 / 0.03²1843
95% 信心0.5（保守）1.961.96²·0.25 / 0.03²1068
📖 用 p=0.5 得到的 n（1843）比用 0.44（1817）大 → 印證「0.5 最保守」。信心從 99% 降到 95%，n 也大幅下降（1843→1068）。

PART 4  ·  母體變異數 σ² 的區間估計

前面估的都是「中心」（μ、p）；有時我們更關心「穩定度、變異程度」——這就要估 σ²。工具換成上週的卡方分配 χ²。因為卡方不對稱，區間的上下限用不同的臨界值。

卡方區間估計 + 房租範例Week 4-13

關鍵樞紐量（pivotal quantity）

點估計 S² = Σ(Xi − X̄)²n − 1
(n−1)S²σ² ≈ χ²n−1

（母體常態，或 n≥30 由中央極限定理）

💡 白話：把「樣本變異數 S² 相對於母體 σ²」的比值放大 (n−1) 倍，會服從自由度 n−1 的卡方分配。因為卡方是右偏、非對稱，所以左右臨界值不一樣。

(1−α)·100% 信賴區間

(n−1)S²χ²α/2, n−1 < σ² ≤ (n−1)S²χ²1−α/2, n−1
⚠️ 容易記反：分母越大→整體越小。所以大的臨界值 χ²α/2 放在左邊（下限），小的臨界值 χ²1−α/2 放在右邊（上限）。要 σ（標準差）就把整條開根號。

🔖 範例：Apartment Rents（估母體標準差）

同房租樣本：n=10，S=$60，母體常態，求 σ 的 95% 信賴區間。df=9，查表得 χ²0.025,9=19.02、χ²0.975,9=2.70。

σ² 下限 = (9)(60²)/19.02 = 32400/19.02 = 1703.47
σ² 上限 = (9)(60²)/2.70 = 32400/2.70 = 12000
→ 1703.47 < σ² ≤ 12000 → 開根號 → 41.27 ≤ σ ≤ 109.54
📖 有 95% 信心，母體月租標準差介於 $41.27 到 $109.54。注意區間不對稱（中心 60 不在正中間），這正是卡方右偏的結果。

PART 5  ·  兩母體差異的估計

商業決策常是「比較」：新舊產品、兩家供應商、活動前後。估的對象變成兩母體平均差 μ₁−μ₂ 或兩母體比率差 p₁−p₂。核心觀念：兩獨立樣本的變異數會相加。再分獨立樣本 / 配對樣本兩種設計。

σ₁, σ₂ 已知：用 ZWeek 4-15

雙尾 (1−α) 信賴區間

(X̄₁−X̄₂) ± Zα/2 · √σ₁²n₁+σ₂²n₂

若已知 σ₁=σ₂=σ，標準誤簡化成 σ·√(1/n₁ + 1/n₂)。單尾則只取上式的一邊（+ 或 −）。

σ 未知：用 S 與 t（含合併變異數）Week 4-16~17
白話：σ 未知就用 S 代替。依「兩母體變異數是否相等」分兩條路，且大樣本(n≥30)用 Z、小樣本用 t。
情況 A

變異數不相等 σ₁ ≠ σ₂

標準誤直接用各自的 S：

√S₁²n₁+S₂²n₂

自由度 v 用 Welch 公式（較複雜，計算機會處理）。大樣本：t≈Z。

情況 B

變異數相等 σ₁ = σ₂ = σ

先算合併變異數（pooled variance）：

Sp² = (n₁−1)S₁² + (n₂−1)S₂²n₁ + n₂ − 2

標準誤 = Sp·√(1/n₁+1/n₂)，自由度 v = n₁+n₂−2。

💡 合併變異數白話：兩組都在估同一個 σ，那就把兩組資訊「加權平均」成一個更可靠的估計（權重是各自的自由度 n−1）。這是情況 B（等變異）專屬。
🔖 範例：Par 高爾夫球（大樣本・σ 未知不等）Week 4-18
樣本1 Par樣本2 Rap
樣本數 n12080
平均 X̄235 碼218 碼
標準差 S1520
點估計 μ₁−μ₂ = 235 − 218 = 17 碼
大樣本(n≥30) → t≈Z=1.96。標準誤 = √(15²/120 + 20²/80) = √(1.875 + 5) = 2.62
17 ± 1.96 × 2.62 → 11.86 ≤ μ₁−μ₂ ≤ 22.14
📖 整個區間都 > 0 → 有 95% 信心 Par 球平均比 Rap 球多飛 11.86～22.14 碼，「extra distance」的宣稱站得住腳。
🔖 範例：Specific Motors 油耗（小樣本・合併變異數）Week 4-19
樣本1 M 車樣本2 J 車
樣本數 n128
平均 X̄29.8 mpg27.3 mpg
標準差 S2.561.81
⚠️ 兩個假設：① 兩母體油耗皆為常態；② 兩母體變異數相等（→ 用合併變異數 + t）。
Sp² = [(12−1)(2.56²) + (8−1)(1.81²)] / (12+8−2) = 5.28
df = n₁+n₂−2 = 18 → t0.025,18 = 2.101；點估計 = 29.8−27.3 = 2.5
2.5 ± 2.101 × √[5.28(1/12+1/8)] → 0.3 ≤ μ₁−μ₂ ≤ 4.7
📖 區間 (0.3, 4.7) 全大於 0 → 有 95% 信心 M 車油耗優於 J 車，差距在 0.3～4.7 mpg。小樣本用 t（2.101）而非 Z（1.96），區間較寬更保守。
配對樣本 Matched SamplesWeek 4-20~21
白話：配對（paired）＝每個受測對象同時提供一對數據（如同一批人「用藥前 vs 用藥後」）。因為每一對來自同一來源，能消掉個體間的差異，抽樣誤差通常比獨立樣本更小（又叫 blocking 集區）。

做法：先算每對的差 d，再對「差」做單母體區間估計

d₁=X₁₁−X₂₁
,
d₂=X₁₂−X₂₂
…
dₙ=X₁ₙ−X₂ₙ
差的平均：d̄ = Σ din
差的標準差：sd = √Σ(di−d̄)²n−1
💡 如何分辨獨立 vs 配對？① X₁ⱼ 與 X₂ⱼ 天生成對、彼此相依 → 配對；② 兩組各自獨立抽、互不相干 → 獨立；③ n₁ ≠ n₂ 一定是獨立樣本（配對要求兩邊筆數相同）。
兩母體比率差 p₁−p₂Week 4-21~22

點估計與標準誤

點估計：p₁−p₂ 的估計 = p̂₁ − p̂₂
σ̂p̂₁−p̂₂ = √p̂₁(1−p̂₁)n₁+p̂₂(1−p̂₂)n₂

大樣本條件：n₁p̂₁、n₁(1−p̂₁)、n₂p̂₂、n₂(1−p̂₂) 都 ≥ 5，可用常態近似。

(1−α) 信賴區間

(p̂₁−p̂₂) ± Zα/2 · √p̂₁(1−p̂₁)n₁+p̂₂(1−p̂₂)n₂
💡 跟兩母體平均差同一個精神：點估計是兩比率相減，標準誤是兩個變異數相加後開根號。
🔖 範例：MRA 廣告成效Week 4-23

廣告活動後：250 戶中 120 戶知道產品（p̂₁）；活動前：150 戶中 60 戶知道（p̂₂）。廣告有沒有提升知名度？（95%）

p̂₁ = 120/250 = 0.48，p̂₂ = 60/150 = 0.40 → 點估計 = 0.08
標準誤 = √[0.48(0.52)/250 + 0.40(0.60)/150] = 0.051
0.08 ± 1.96 × 0.051 = 0.08 ± 0.10 → −0.02 ≤ p₁−p₂ ≤ 0.18
⚠️ 結論的細節：區間 (−0.02, 0.18) 包含 0（紅色虛線）。雖然點估計是正的 +0.08，但因為區間跨過 0，在 95% 信心下還不能斷定廣告一定有效——差異有可能只是抽樣誤差。

PART 6  ·  F 檢定：比較兩母體變異數

前面用兩母體平均差時，我們常要先假設「σ₁=σ₂ 還是不等」。到底怎麼判斷？——比較兩個樣本變異數的比值 S₁²/S₂²，它服從F 分配。

F 分配・S₁²/S₂²Week 4-24~25
白話：要回答「兩母體變異數相不相等」，最自然是看兩個樣本變異數的比值。若真的相等，比值應該接近 1。這個比值服從 F 分配（由「分子自由度 u、分母自由度 v」兩個參數決定）。假設前提：兩母體都常態。

F 統計量與性質

S₁²/σ₁²S₂²/σ₂² ≈ Fn₁−1, n₂−1

母體常態時：S₁²/S₂² ≈ (σ₁²/σ₂²)·Fn₁−1,n₂−1

  • u = 分子自由度（n₁−1）；v = 分母自由度（n₂−1）。
  • 右偏、只取正值（比值不會為負）。
  • 倒數性質：Fα;u,v = 1 / F1−α;v,u。
  • 與 t 的關係：[tv]² = F1,v。

讀圖：F 分配（右偏）

📖 讀圖：F 分配集中在 1 附近但拖著長長的右尾。若 S₁²/S₂² 落在 1 附近→兩變異數可能相等；若明顯偏離 1（落到右尾臨界值外）→可判定不相等。查 F 表需同時定「分子、分母自由度」與顯著水準（.05 / .025 / .01）。
