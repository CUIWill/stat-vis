PART 1  ·  連續機率分配基礎

上週的隨機變數「數得出來」（離散）；本週的變數可以是區間內任何值（連續）。關鍵轉變：機率不再看「某個點」，而是看「曲線下的面積」。

連續隨機變數 Continuous Random VariableWeek 3-1
白話：一個可以取「區間內任何值」的變數，例如時間、身高、重量。跟上週離散變數最大的不同：它的可能值數不完（兩個數字之間永遠還塞得下更多數字）。
核心特性①

取值連續

可以是實數線上某個區間、或好幾個區間裡的任何值。例：等公車時間可以是 3.14159… 分鐘。

核心特性②

單點機率 = 0

問「剛好等於某個值」的機率沒有意義，一律是 0。因為可能值有無限多個，分給任何單一點的機率都是 0。
P{X = x} = 0

核心特性③

只談機率密度

給定一個值，我們只知道它的機率密度（probability density） f(x)，而不是機率本身。

機率密度函數 f(x)（讀圖方法：高度＝密度，不是機率！）

📖 讀圖：縱軸是「密度」不是機率。曲線越高的地方，代表 X 落在那附近的可能性越大，但單一點本身機率仍是 0。要拿到機率，得看下一節「一段區間下的面積」。
機率 = 曲線下的面積Week 3-1

某區間的機率＝那段面積

X 落在 x₁ 到 x₂ 之間的機率，等於機率密度曲線在這兩點之間圍出的面積（積分）。

P{x₁ ≤ X ≤ x₂} = x₂x₁∫ f(x) dx
💡 ∫（積分符號）不用怕：它就是「把一段面積加總起來」的意思。EMBA 不會叫你手算積分，交給查表或計算機。

兩條鐵則（跟離散版對應）

① 密度不為負：f(x) ≥ 0（曲線不會跑到 x 軸下面）
② 全部面積 = 1：∫ f(x) dx = 1（整條曲線下總面積 = 1）

跟離散的 Σf(x)=1 是同一個精神，只是把「加總」換成「面積」。

期望值與變異數（把 Σ 換成 ∫）

期望值（平均、中心）
E(X) = ∫ x·f(x) dx
變異數（分散程度）
Var(X) = ∫ (x−μ)²·f(x) dx
📖 對照上週：離散是 E(X)=Σx·f(x)，連續是 E(X)=∫x·f(x)dx。「加權平均」的概念完全一樣，只是離散用加總、連續用面積。

PART 2  ·  三大連續分配

均勻（平的長方形）、常態（鐘形，最重要）、指數（等待時間、遞減）。每個分配都看三件事：中心（平均）、分散（變異數）、形狀。

均勻分配 Uniform Distribution（均勻分配）Week 3-2
白話：在 a 到 b 之間每個位置的可能性都一樣，機率跟「區間長度」成正比。密度曲線是一條平的橫線，圍出來的圖形是長方形。

機率密度函數 f(x)

f(x) = 1b − a （a ≤ x ≤ b），否則 = 0

a＝變數能取的最小值；b＝最大值。高度固定為 1/(b−a)，讓長方形總面積剛好 = 1。

累積 F(x)：x<a 時 0；a≤x≤b 時 (x−a)/(b−a)；x>b 時 1

讀圖方法：平坦的長方形

📖 高度一樣＝每個值機率密度相同。要某段機率，就量那段長方形面積＝(長度)×(1/(b−a))。

三件事：中心・分散・形狀

① 中心 Mean
E(X) = a + b2

就是頭尾的正中間

② 分散 Variance
Var(X) = (b − a)²12

區間越寬越分散

③ 形狀 Shape
▭

平的長方形

課本例：統計成績 X ~ Uniform(70, 90)
E(X) = (70+90)/2 = 80
Var(X) = (90−70)²/12 = 400/12 ≈ 33.33
標準差 σ = √33.33 ≈ 5.774
P{X ≥ 85} = (90−85)/(90−70)
= 5/20 = 0.25
（課本另算 P 到 85 為 15/20=0.75）
均勻分配範例：Slater 沙拉秤重Week 3-3
情境：Slater 餐廳按沙拉重量收費。抽樣發現取的沙拉重量在 5 到 15 盎司之間均勻分布。→ a=5, b=15。

建立密度與平均、變異數

f(x) = 1/(15−5) = 1/10（5 < x < 15）
E(X) = (5+15)/2 = 10 盎司
Var(X) = (15−5)²/12 = 100/12 ≈ 8.33 → σ ≈ 2.89

P(12 < X < 15)＝那段長方形面積

P(12<X<15) = (15−12)×(1/10) = 0.3
常態（高斯）分配 Normal Distribution（常態分配）Week 3-4
白話：最重要、最常見的連續分配，長得像一口對稱的鐘。很多自然現象（身高、體重、測量誤差）都近似常態。只要兩個數字就能完全決定它：μ（平均，決定位置）和 σ（標準差，決定胖瘦）。

常態機率密度函數（看看就好，不用背）

f(x) = 1σ√(2π) e−(x−μ)²/(2σ²)

其中 π ≈ 3.14159、e ≈ 2.71828 是固定常數。式子很嚇人，但實務上一律查表或用計算機，不會手算。

💡 兩個參數的角色：μ 把整個鐘平移到哪（中心）；σ 決定鐘有多胖多矮（越大越扁平）。

讀圖方法：對稱的鐘形

📖 最高點在正中央（μ）。左右完全對稱，所以平均 = 中位數 = 眾數，全都在 μ。
① 中心

E(X) = μ

最高點的位置就是平均數。

② 分散

Var(X) = σ²

σ 越大鐘越寬越矮；越小越尖越窄。

③ 形狀

對稱鐘形

曲線下總面積 = 1（左右各 0.5）。

常態的性質・參數的影響Week 3-5

四個必記性質

① 鐘形且左右對稱
② 平均 = 中位數 = 眾數 = μ
③ μ 和 σ 完全決定整條曲線
④ 反曲點（曲線凹凸轉折）在 μ ± σ
💡 反曲點（inflection point）＝曲線從「向下彎」變成「向上彎」的地方，剛好落在離中心一個標準差 μ±σ 之處。這是 σ 的幾何意義。

參數變化的效果（讀圖方法：同軸比較胖瘦位置）

📖 三條鐘中心都在同一點（μ 相同）。σ 越小越集中（高瘦），σ 越大越分散（矮胖）。若改變 μ，整條鐘會左右平移。
⚠️ 常態機率＝曲線下面積。想求 P{X ≤ x} 得算積分——但這式子沒有簡單解，得靠查表，而這正是下一節「標準化」要解決的問題。
標準化與 Z 分數（一張表通吃）Week 3-6

問題：每種 μ、σ 都要一張表 → 無限多張表！

「我不會積分，能給我一張表查嗎？」——問題是 μ、σ 有無限多組合，難道要印無限多張表？解法：先把任何常態變成同一個標準版本。

標準化定理

任何常態都能變成「標準常態」

若 X ~ Normal(μ, σ²)，令 Z = X − μσ，則 Z ~ Normal(0, 1)
💡 Z 分數（Z-score）的白話：把原始值換算成「距離平均幾個標準差」。Z=0 代表剛好在平均；Z=1 代表比平均高一個標準差；Z=−2 代表比平均低兩個標準差。單位被消掉了，所以任何常態都能用同一把尺。

讀圖方法：把任意常態「壓」成標準常態

📖 左邊每個 x 都用 Z=(x−μ)/σ 換算，就映射到右邊的標準常態。只需一張標準常態表就能查所有機率。
標準常態與經驗法則 68-95-99.7Week 3-7
標準常態 Standard Normal：平均 0、標準差 1 的常態，用字母 Z 表示。Z 就是「x 離平均幾個標準差」的度量。

讀圖方法：離平均 1、2、3 個 σ 各涵蓋多少（面積）

±1σ
P(μ−σ < X < μ+σ)
= P(−1<Z<1)
≈ 68.26%
±2σ
P(−2<Z<2)
≈ 95.46%
±3σ
P(−3<Z<3)
≈ 99.73%
💡 對稱好用的公式：P(Z ≤ −z) = P(Z ≥ z) = 1 − P(Z ≤ z)。查表只需查正的一半，另一半用對稱推出來。（Z 表在課本附錄 TABLES.pdf）
常態分配範例：X ~ Normal(40, 6²)Week 3-9
反查：已知機率求 x

P{X ≤ x} = 0.45，求 x？

先查表：P{Z ≤ z}=0.45 → z ≈ −0.13
還原：x = μ + zσ = 40 + (−0.13)(6) ≈ 39.22
📖 機率小於 0.5 → z 為負 → x 落在平均 40 的左邊，合理。
正查：已知 x 求機率

P{X ≥ 46.48} = ?

標準化 z = (46.48 − 40)/6 = 1.08
P{Z ≥ 1.08} = 1 − P{Z ≤ 1.08}
= 1 − 0.86 = 0.14
💡 解題三步驟：①標準化算 z ②查標準常態表得面積 ③依題意（≥、≤、之間）用對稱與 1− 調整。
指數分配 Exponential Distribution（指數分配）Week 3-10
白話：描述「等一件事發生要多久」的分配（例如下一位客人多久後上門、機器多久後故障）。和上週的布阿松分配是一對搭檔：布阿松數「一段時間內發生幾次」，指數量「距離下一次要等多久」。

密度與累積函數

f(x) = λe−λx（x ≥ 0）
累積 F(x) = 1 − e−λx（x ≥ 0），否則 0

λ（lambda）＝單位時間的平均發生率；x＝等待時間。
註：課本用 λ 表示「平均等待時間」寫法，下方三件事採此定義。

讀圖方法：一路遞減的曲線

📖 x=0 時最高，之後單調下滑、永不觸底。等越久還沒發生的機率越低。
① 中心 Mean
E(X) = 1λ
② 分散 Variance
Var(X) = 1λ²
③ 形狀 Shape

PART 3  ·  統計推論與點估計

前面都在「已知母體、算機率」（演繹）。從這裡開始反過來：只看到一小把樣本，反推整個母體長怎樣（歸納 = 推論）。

統計推論的基本詞彙Week 3-11
統計推論 Statistical Inference：從「樣本」帶的資訊，去推估「母體」的特性。因為通常沒辦法（也不划算）調查全部，只能抽一部分來估。
母體 Population

全部對象

你關心的所有元素的集合。例：全校 1500 位申請人。

樣本 Sample

母體的子集

從母體抽出的一小部分。例：隨機抽 50 位申請人。

參數 Parameter

母體的數字特徵

描述母體的數值，如母體平均 μ、母體比率 p。通常未知，是我們想估的目標。

估計 Estimate

用樣本算出的近似值

樣本結果只是母體特徵的估計，不會百分百等於真值。

讀圖方法：兩個方向（演繹 vs 推論）

母體 Population
參數 μ, σ, p
機率論・演繹 ↓
↑ 推論・歸納
樣本 Sample
統計量 X̄, S, p̂
📖 往下（機率論／演繹）：已知母體，算樣本會怎樣。往上（統計推論／歸納）：只看到樣本，反推母體。本週後半都在走「往上」這條路。
隨機樣本 Random Sample

抽取時沒有系統性偏向，每個樣本被選中的機率都相等。這是好估計的前提。

點估計 Point Estimation

用樣本資料算出一個數字去估母體參數：X̄→μ、S²→σ²、S→σ、p̂→p。

抽樣誤差與普查 vs 抽樣Week 3-12
抽樣誤差 Sampling Error：不偏點估計值與母體真值之間的「絕對差距」。例如樣本平均 X̄ 與母體平均 μ 的差 |X̄ − μ|。它源自「只用一部分（樣本）而非全部（母體）」。

四種常見抽樣誤差

樣本平均：|X̄ − μ|
樣本比率：|p̂ − p|
樣本標準差：|S − σ|
樣本變異數：|S² − σ²|
課本例：NTUST 每年 1,500 份申請

想知道：申請人平均 SAT 分數（μ）＆本地生比率（p）

方案①：普查 Census（全部 1500 人）
母體平均 μ = 990
母體標準差 σ = 80
母體比率 p = 1080/1500 = 0.72

普查得到「真值」，但費時費錢。

方案②：抽樣 Sample（隨機 50 人）
X̄ = 49850/50 = 997（估 μ）
S ≈ 75.2（估 σ）
p̂ = 34/50 = 0.68（估 p）

快又省，但有抽樣誤差。

誤差：|X̄−μ|=|997−990|=7　|S−σ|=|75.2−80|=4.8　|p̂−p|=|0.68−0.72|=0.04
點估計：用統計量估參數Week 3-12 ~ 3-13

對照表：樣本統計量 → 母體參數

母體參數（想估的目標，未知）樣本統計量（點估計式）NTUST 數值
母體平均 μ樣本平均 X̄ = ΣXᵢ / n997（估 990）
母體標準差 σ樣本標準差 S = √[Σ(Xᵢ−X̄)²/(n−1)]75.2（估 80）
母體比率 p樣本比率 p̂ = X / n0.68（估 0.72）
💡 樣本標準差 S 的分母是 n−1 不是 n（叫「自由度」修正），這樣算出來對 σ 才不會系統性偏低。
抽樣分配 Sampling Distribution 是什麼Week 3-13
關鍵觀念：樣本統計量（如 X̄）本身也是隨機變數——換一組樣本就會得到不同的 X̄。把「所有可能樣本算出的統計量」的機率分配，就叫抽樣分配。

讀圖方法：一個母體 → 抽很多次 → 每次得一個統計量

📖 每抽一組樣本就得到一個 X̄；X̄ 們自己也形成一個分配。研究這個分配的中心、分散、形狀，就能量化「用樣本估母體有多準」。

PART 4  ·  抽樣分配（X̄、p̂、S²）

三個最重要的抽樣分配。X̄ 靠中心極限定理變常態、p̂ 也近似常態、S² 走卡方分配。每個都看：中心在哪、分散多少、形狀是什麼。

樣本平均數 X̄ 的抽樣分配Week 3-14
中心（期望值）

X̄ 平均起來 = 母體平均

μX̄ = E(X̄) = μ

X̄ 是「不偏」的：長期平均剛好命中 μ，不會系統性偏高或偏低。

分散（標準誤）

X̄ 的標準差 = σ ÷ √n

σX̄ = σ√n

叫「標準誤」。n 越大，X̄ 越集中在 μ 附近——樣本越多估得越準。

有限母體要乘「有限修正因子」

σX̄ = σ√n × √N − nN − 1
💡 什麼時候可以忽略修正？當 n/N < 0.05（樣本占母體不到 5%），就把母體當作無限大，直接用 σ/√n 即可。√[(N−n)/(N−1)] 就是「有限修正因子」。
中心極限定理 Central Limit Theorem（中心極限定理）Week 3-15
白話（本週最重要定理）：不管母體原本長什麼樣（歪的、平的都行），只要樣本夠大（n ≥ 30），樣本平均 X̄ 的抽樣分配就會近似常態。這就是為什麼常態分配到處都用得上。

結論公式

X̄ ~ Normal(μ, σ²n)
Z = X̄ − μσ/√n ~ Normal(0, 1)
n ≥ 30（大樣本）

CLT 生效，X̄ 直接當常態用，母體形狀不重要。

n < 30（小樣本）

只有在母體本身就是常態時，X̄ 才能當常態用。

讀圖方法：樣本越大，X̄ 的分配越窄越像鐘

📖 中心都在 μ。n 越大，標準誤 σ/√n 越小，X̄ 越緊縮在 μ 周圍 → 估計越準。
X̄ 抽樣分配範例（NTUST）Week 3-16 ~ 3-17

設定：μ=990, σ=80, 抽 n=50

標準誤 σX̄ = 80/√50 = 11.3

n=50 ≥ 30 → 由 CLT，X̄ ~ Normal(990, 11.3²)。

問：抽 50 人，X̄ 落在 μ ± 10（即 980~1000）的機率？
標準化：z = (980−990)/11.3 = −0.88；z = (1000−990)/11.3 = +0.88
P{980 ≤ X̄ ≤ 1000} = P{−0.88 ≤ Z ≤ 0.88} = 0.6212
📖 讀法：隨機抽 50 人，樣本平均落在真值 ±10 分內的機會約 62%。想更準（提高這機率），就加大 n 讓 σX̄ 縮小。
樣本比率 p̂ 的抽樣分配Week 3-17 ~ 3-18
白話：把「比率」也拿來估。樣本裡「成功」的比例 p̂ = X/n（X 是成功次數，服從二項分配）。跟 X̄ 一樣，p̂ 也有自己的抽樣分配。
中心
μp̂ = E(p̂) = p

p̂ 也是不偏估計，平均起來命中母體比率 p。

分散（標準誤）
σp̂ = √p(1−p)n

有限母體再乘 √[(N−n)/(N−1)]，規則同 X̄。

形狀：什麼時候能當常態？

np ≥ 5（且 n(1−p) ≥ 5）時可用常態近似

p̂ ~ Normal(p, p(1−p)n)
⚠️ 若 np < 5，常態近似不夠好，得回頭用二項分配硬算：P{p̂ ≤ a} = P{X ≤ na} = Σ C(n,k)pᵏ(1−p)ⁿ⁻ᵏ。
p̂ 抽樣分配範例（NTUST）Week 3-19

設定：p=0.72, n=50

檢查 np = 50×0.72 = 36 ≥ 5 ✓ → 可用常態近似
標準誤 σp̂ = √[0.72×0.28/50] = √0.004032 ≈ 0.0635
問：抽 50 人，p̂ 落在 p ± 0.05（即 0.67~0.77）的機率？
標準化：z = (0.67−0.72)/0.0635 = −0.79；z = (0.77−0.72)/0.0635 = +0.79
P{0.67 ≤ p̂ ≤ 0.77} = P{−0.79 ≤ Z ≤ 0.79} = 0.5704
樣本變異數 S² 與卡方分配Week 3-20 ~ 3-21
白話：估「分散程度」時，樣本變異數 S² 的抽樣分配不是常態，而是走一個新分配——卡方分配 χ²（chi-square）。它右偏（右邊拖一條長尾），且只出現在正值。
卡方分配怎麼來的

把 n 個獨立的標準常態 Z 各自平方再相加：
Z₁²+Z₂²+…+Zₙ² ~ χ²(自由度 n)。

E(χ²)=ν，Var(χ²)=2ν（ν=自由度）
S² 的抽樣分配（母體須常態）
(n−1)S²σ² ~ χ²(n−1)

自由度 = n−1。把 S² 換算成這個量，就能查卡方表。

讀圖方法：右偏、只在正值的卡方曲線

📖 卡方不對稱：左邊被 0 擋住，右邊拖長尾。自由度越大，越往右移、越接近對稱的鐘形。
卡方分配範例（NTUST）Week 3-23

設定：σ=80, 抽 n=51（自由度 = 50）

問：樣本標準差 S 落在 61.664 ~ 92.955 的機率？
先把 S 轉成卡方量（平方後乘 (n−1)/σ²）：
下界：50×(61.664)²/80² = 29.707
上界：50×(92.955)²/80² = 67.505
P{29.707 ≤ χ²₅₀ ≤ 67.505} ≈ 0.99 − 0.05 = 0.94
📖 三步驟同常態，只是查的是「卡方表」而非「Z 表」：①把 S 換算成 (n−1)S²/σ² ②查卡方表得兩端機率 ③相減得區間機率。
