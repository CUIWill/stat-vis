PART 1  ·  機率基礎

機率是什麼？怎麼數出所有可能結果（計數法則、排列、組合），又怎麼把機率分配給每個結果。

機率導論：Probability 是什麼？Week 2-1
核心定義

機率 Probability

一個數字，用來衡量某件事（事件）發生的可能性有多高。

範圍

永遠在 0 到 1 之間

0 = 幾乎不可能；1 = 幾乎一定發生；0.5 = 發生與不發生一樣可能。

白話比喻

像「信心刻度尺」

把「我覺得會不會發生」量化成 0~1 的一把尺，數字越大越有把握。

機率刻度尺（讀圖方法：越往右越可能發生）

實驗、樣本空間、樣本點Week 2-1

實驗 Experiment（實驗）

任何會產生「明確結果」的過程。
例：擲一顆骰子。

樣本空間 Sample Space（樣本空間）

記作 S，是「所有可能結果」的集合。

樣本點 Sample Point

樣本空間裡的一個結果，例：骰子出現「3」。

例：擲一顆骰子，哪個 S 比較好？

S = {1,2,3,4,5,6}   還是   S = {偶, 奇}？
💡 原則：選「能給你最多資訊」的樣本空間。{1,2,3,4,5,6} 比 {偶,奇} 保留更多細節，所以比較好。

S 也可以用條件描述

S = { x | x 是人口超過 100 萬的城市 }
S = { (x, y) | x² + y² ≤ 4 }

「|」唸作「使得 / such that」，左邊是元素長相，右邊是它要滿足的條件。

計數法則：乘法法則與樹狀圖Week 2-1 ~ 2-2
廣義乘法法則 Generalized Multiplication Rule

分 k 個步驟，每步可能數相乘

總結果數 = n₁ × n₂ × n₃ × … × nk

第 1 步有 n₁ 種、第 2 步有 n₂ 種…把每一步的可能數「連乘」就是全部結果數。

💡 白話：做一件事要分好幾關，每關有幾條岔路，全部走法就是各關岔路數相乘。

例：用 1,2,5,6,9 排出幾個「3 位數偶數」？（每個數字只用一次）

個位（須偶）：只有 2,6 → 2 種
百位：剩 4 個 → 4 種
十位：剩 3 個 → 3 種
總數 = 4 × 3 × 2 = 24 個
💡 訣竅：有限制的位置（個位要偶數）先決定，再排其他位置。

樹狀圖 Tree Diagram（Bradley 投資）

兩支股票各有可能漲跌，畫成「分岔」一路展開，葉子就是所有結果。

排列 Permutation（排列）— 有順序Week 2-3
白話：排列在意「順序」。把人排成一排照相，AB 和 BA 算不一樣。

N 個相異物件全部排

N! = N(N−1)(N−2)…(2)(1)

「!」叫階乘。規定 1! = 1，0! = 1。

例：5 個不同數字排 5 位數 = 5! = 120

N 取 n 個來排

PnN = N!(N − n)! = N(N−1)…(N−n+1)
例：5 個數字取 2 位數 = 5×4 = 20

有重複物件的排列（去掉重複算法）

N 個東西中，有 n₁ 個一樣、n₂ 個一樣…時，相異排法為：

N!n₁! n₂! … nk!
例：1,1,5,5,9 排 5 位數 = 5! / (2!·2!) = 120 / 4 = 30
💡 為什麼要除？因為兩個「1」交換你看不出差別，這些重複的排法要除掉。
組合 Combination（組合）— 不管順序Week 2-4
白話：組合不管順序，只看「選到誰」。從一群人選 2 個代表，AB 和 BA 是同一組。

N 取 n 的組合數

CnN = (Nn) = N!n!(N − n)!
💡 排列 ÷ n! = 組合。因為排列把 n 個被選中的東西「又排了順序」，組合不要順序，所以再除掉 n!。
例：撲克牌 Black Jack

52 張抽 2 張的機率

總抽法 = C(52,2) = 1326
Black Jack：1 張 A(4 張) × 1 張 10點(16 張) = 4×16 = 64
P(Black Jack) = 64 / 1326 ≈ 4.8%
例：樂透 Lottery（樂透）

42 選 6 的頭獎機率

總組合 = C(42,6) = 5,245,786
頭獎機率 = 1 / 5,245,786 ≈ 0.000019%
⚠️ 直觀體會：中頭獎比被雷劈還難很多。組合數一大，機率就極小。
指定機率的三種方法Week 2-5 ~ 2-6
① 古典法 Classical

假設每個結果一樣可能

機率 = 有利結果數 / 總結果數。
例：公正骰子每面 = 1/6。

② 相對次數法 Relative Frequency

看歷史/實驗資料

用「過去發生比例」當機率。
例：過去 200 天賣電視的紀錄。

③ 主觀法 Subjective

靠判斷與經驗

環境快速變化、沒歷史資料時，用專家「信念程度」估計。

機率的基本要求（兩條鐵則）

① 每個機率介於 0 與 1：
0 ≤ P(Eᵢ) ≤ 1
② 全部加起來等於 1：
P(E₁)+P(E₂)+…+P(En) = 1
💡 主觀法常與古典/相對次數法「混搭」，得到的估計往往最好。

PART 2  ·  事件與機率關係

把結果打包成「事件」，再用集合運算（互補、聯集、交集）與條件、獨立、貝氏定理算機率。

事件與其機率Week 2-6
事件 Event：樣本空間的一個「子集合」，也就是把幾個樣本點打包起來。某事件的機率 = 它包含的所有樣本點機率相加。
事件 M：Markley 獲利
M = {(10,8),(10,2),(5,8),(5,2)}
P(M) = 0.20+0.08+0.16+0.26 = 0.70
事件 C：Collins 獲利
C = {(10,8),(5,8),(0,8),(20,8)}
P(C) = 0.20+0.16+0.10+0.02 = 0.48
互補・聯集・交集（集合三兄弟）Week 2-7 ~ 2-8
互補 Complement（互補事件）

Aᶜ = 不在 A 裡的全部

P(Aᶜ) = 1 − P(A)
聯集 Union（聯集）

A ∪ B = 在 A 或 B（或都在）

藍色塗滿＝聯集（任一邊都算）

交集 Intersection（交集）

A ∩ B = 同時在 A 和 B

只有中間重疊才算交集

例：M ∪ C（Markley 或 Collins 獲利）

P(M∪C) = 0.20+0.08+0.16+0.26+0.10+0.02 = 0.82

例：M ∩ C（兩者都獲利）

M∩C = {(10,8),(5,8)} → 0.20+0.16 = 0.36
加法法則 Addition LawWeek 2-8 ~ 2-9

算「A 或 B」發生的機率

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
💡 為什麼要減 P(A∩B)？因為中間重疊的部分被「A 算一次、B 又算一次」，多算了一次要扣回來。
驗證 Bradley：P(M∪C) = 0.70 + 0.48 − 0.36 = 0.82 ✓（和直接相加一樣）
互斥事件 Mutually Exclusive（互斥事件）

兩事件「沒有共同樣本點」

一個發生，另一個就不可能發生 → 沒有重疊 → P(A∩B)=0。

P(A ∪ B) = P(A) + P(B)

互斥時加法法則少了那一項，直接相加即可。

條件機率 Conditional Probability（條件機率）Week 2-9 ~ 2-10
白話：「已經知道 B 發生了，那 A 發生的機率是多少？」記作 P(A|B)，唸「在 B 條件下的 A」。新資訊會縮小樣本空間。

公式

P(A | B) = P(A ∩ B)P(B) （須 P(B) > 0）
💡 讀法：分母換成「已知條件 B」的機率，等於把整個世界限縮在 B 之內，再看 A 佔多少。

例：P(C | M) — 已知 Markley 獲利，Collins 也獲利的機率

P(C|M) = P(C∩M) / P(M) = 0.36 / 0.70 ≈ 0.51

原本 P(C)=0.48；知道 M 獲利後升到 0.51 → 兩者略有正關聯。

乘法法則 Multiplication LawWeek 2-10

算「A 且 B」同時發生（交集）

P(A ∩ B) = P(B)·P(A|B) = P(A)·P(B|A)

把條件機率公式兩邊乘回 P(B) 就得到它。先發生一件、再在其條件下發生另一件。

驗證：P(M∩C) = P(M)·P(C|M) = 0.7 × 0.51 ≈ 0.36 ✓
💡 多事件連鎖版：P(E₁∩E₂∩…∩En) = P(E₁)·P(E₂|E₁)·P(E₃|E₁∩E₂)···一路接龍。
獨立事件 Independent Events（獨立事件）Week 2-11
白話：A 發生與否，完全不影響 B 的機率 → 兩者獨立。判斷標準：P(A|B)=P(A)。

獨立的乘法法則（變簡單了）

P(A ∩ B) = P(A)·P(B)

獨立時不用條件機率，直接相乘。多事件：全部機率連乘。

反過來也能「檢驗」是否獨立

P(M∩C)=0.36，但 P(M)·P(C)=0.7×0.48=0.336
⚠️ 0.36 ≠ 0.336 → M 與 C 不獨立（兩股票走勢有關聯）。
貝氏定理 Bayes' Theorem（貝氏定理）Week 2-11 ~ 2-13

核心想法：用新資訊「更新」舊機率

事前機率
Prior
→
新資訊
New Info
→
事後機率
Posterior

事前機率（事前機率 Prior）= 還沒拿到證據前的猜測；事後機率（事後機率 Posterior）= 看到證據後修正的結果。

全機率定理 Theorem of Total Probability

B 可能透過好幾條互斥路徑 A₁,A₂… 發生，把每條路的機率加起來：

P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + …

貝氏定理公式

P(Aᵢ|B) = P(Aᵢ)·P(B|Aᵢ) Σ P(Aⱼ)·P(B|Aⱼ)

適用條件：A₁,A₂… 互斥，且聯集是整個樣本空間。

例：兩條路徑 A₁, A₂（課本數字）

事前條件機率P(Aᵢ∩B)
P(A₁)=.7P(B|A₁)=.2.14
P(A₂)=.3P(B|A₂)=.9.27
P(B)=.41
P(A₁|B) = .14 / .41 ≈ 0.34
P(A₂|B) = .27 / .41 ≈ 0.66
💡 經典應用「選擇題」：學生會的機率 p、用猜的 1−p、每題 m 個選項。答對的條件下「真的會」的機率 = p / [p + (1−p)/m] = mp / [1+(m−1)p]。

PART 3  ·  隨機變數與機率分配

把實驗結果變成數字（隨機變數），用機率分配描述它，再算它的中心（期望值）與分散（變異數）。

隨機變數 Random Variable（隨機變數）Week 2-14 ~ 2-15
白話：把「實驗結果」翻譯成一個數字。例如擲兩顆骰子，把結果對應成「點數和」X。它是一個從樣本空間 S 對應到實數 R 的「對照表」。

讀圖方法：把結果 → 數字（map X: S → R）

離散 Discrete（離散隨機變數）

數得出來的值

有限個或可數的值。例：一天賣出電視 0,1,2,3,4 台；一天來客數 0,1,2,…（無上限但仍可數）。

連續 Continuous（連續隨機變數）

區間中任何值

可取某區間內任意數值。例：身高、時間、重量（本週聚焦離散，連續下週）。

離散機率分配 Discrete Probability DistributionWeek 2-15 ~ 2-16
白話：機率分配＝告訴你「每個值各分到多少機率」的分配表。用機率函數 f(x) 表示每個 x 的機率。

離散機率函數的兩個必要條件

① 每個機率 ≥ 0：f(x) ≥ 0
② 全部加起來 = 1：Σ f(x) = 1

例：電視店一天銷量 X 的分配表

f(x) = 該銷量天數 ÷ 200（相對次數法）。

期望值與變異數（離散）Week 2-16 ~ 2-17
期望值 Expected Value（期望值）

分配的「中心 / 平均」

E(X) = μ = Σ x·f(x)

把每個值 × 它的機率，全部加起來＝「長期平均會落在哪」。

變異數 Variance（變異數）

分配的「分散程度」

Var(X) = σ² = Σ(x−μ)²·f(x)
= Σx²·f(x) − μ²　(快算版)

標準差 σ = √Var(X)，把分散程度換回原單位。

例：電視銷量 X 的 E(X)、Var(X)（逐欄計算）

E(X) = 1.2 台
Var(X) = 1.66
σ = √1.66 ≈ 1.29 台

PART 4  ·  常見的離散機率分配

七種常用模型：均勻、伯努利、二項、布阿松、超幾何、幾何、負二項（＋補充多項式）。每個都附「用在什麼情境」與公式計算機。

離散均勻分配 Discrete UniformWeek 2-18
用在哪：每個值「機率都一樣」。例：公正骰子 1~6，每面 1/6。

機率函數

f(x) = 1n　x = x₁,…,xn

n = 可能值的個數。每格一樣高。

μ = (1/n)Σxᵢ　σ² = (1/n)Σ(xᵢ−μ)²

形狀（讀圖方法：每柱等高）

伯努利試驗 Bernoulli TrialWeek 2-19
用在哪：只做一次、只有兩種結果的試驗 — 成功(1, 機率 p) 或失敗(0, 機率 q=1−p)。例：投一次硬幣。

期望值與變異數

μ = E(X) = p
σ² = Var(X) = p·q = p(1−p)
伯努利過程 Bernoulli Process

把伯努利試驗重複 n 次（四條件）

① n 次重複試驗　② 每次只有兩種結果　③ 成功機率 p 固定不變　④ 每次互相獨立。
→ 滿足這四點，就能用下面的二項式分配。

二項式分配 Binomial Distribution（二項式分配）Week 2-19 ~ 2-21
用在哪：n 次獨立伯努利試驗中，「成功幾次」。例：抽 3 個產品，有幾個不良品。參數：n（次數）、p（單次成功機率）。

機率函數

f(x) = (nx) pˣ qn−x　x = 0,1,…,n

C(n,x) = 在 n 次中挑出哪幾次成功的組合數；pˣ = 那幾次成功；q^(n−x) = 其餘失敗。

μ = E(X) = np
σ² = Var(X) = npq

例：3 個產品、不良率 10%，恰好 1 個不良？

n=3, p=0.1, 求 f(1)
f(1)=C(3,1)·0.1¹·0.9² = 3×0.1×0.81 = 0.243
E(X)=3×0.1=0.3　σ²=3×0.1×0.9=0.27

形狀隨 p 改變（讀圖：p<.5 右偏、p=.5 對稱、p>.5 左偏）

⚠️ 眾數位置：若 (n+1)p 不是整數，最高的那格在 x = ⌊(n+1)p⌋；機率隨 x 先升後降。
布阿松分配 Poisson Distribution（布阿松分配）Week 2-22 ~ 2-23
用在哪：固定時間/區域內某事件「發生幾次」，而且各區間獨立、平均率固定。例：一小時內幾班公車經過。參數：λ（平均發生次數）。

機率函數

f(x) = λˣ e−λx!　x = 0,1,2,…　e ≈ 2.71828
μ = E(X) = λ
σ² = Var(X) = λ　(平均=變異數，特色！)

例：每 10 分鐘一班公車，下一小時少於 3 班的機率？

λ = 60/10 = 6（每小時）
P(X<3)=f(0)+f(1)+f(2) = e⁻⁶(1+6+18) ≈ 0.0620
二項 → 布阿松 的近似

n 很大、p 很小時

當 n→∞、p→0 而 np 維持定值，可用 λ = np 的布阿松來近似二項分配（算起來省力很多）。

超幾何分配 Hypergeometric（超幾何分配）Week 2-24 ~ 2-25
用在哪：從 N 個有限母體中「不放回」抽 n 個，數成功幾個。因為不放回，每抽一次成功機率會變（試驗不獨立）。

機率函數

f(x) = (rx)(N−rn−x) (Nn)

N=母體總數，r=母體中成功個數，n=抽樣數，x=樣本中成功數。x = 0,1,…,min(n,r)。

μ = n·(r/N)
σ² = n·(r/N)·(1−r/N)·(N−n)/(N−1)

例：52 張抽 5 張，3 紅 2 黑的機率（不放回）

C(26,3)·C(26,2) / C(52,5) ≈ 0.3251

對照「放回」版本（二項）= C(5,3)·0.5⁵ = 0.3125，兩者略有差異。

超幾何 → 二項 的近似

當 n ≪ N（抽很少）

母體夠大、抽得夠少時，放不放回差不多，可用 p = r/N 的二項近似。

幾何分配 Geometric Distribution（幾何分配）Week 2-26 ~ 2-27
用在哪：一直做獨立試驗，問「第幾次才第一次成功」。例：第幾次買樂透才第一次中。參數只有 p。

機率函數

f(x) = p(1−p)x−1 = p·qx−1

x = 1,2,…：前 x−1 次都失敗、第 x 次成功。

μ = 1/p
σ² = (1−p)/p²

例：每次中獎率 0.02722，第 4 次才首次中？

f(4)=0.02722×0.97278³ ≈ 0.025
平均要買 E(X)=1/0.02722 ≈ 36.7 次
負二項（Pascal）分配 Negative Binomial（負二項分配）Week 2-27 ~ 2-28
用在哪：幾何分配的延伸 — 問「第幾次才達到第 r 次成功」。當 r=1 時就退化成幾何分配。

機率函數

f(x) = (x−1r−1) pr(1−p)x−r

x = r, r+1, …：第 x 次拿到第 r 次成功。

μ = r/p
σ² = r(1−p)/p²

直覺：把它拆成 r 段幾何

第 r 次成功 = 第 1 次成功 + 第 2 次… 共 r 段，每段都是幾何分配（平均 1/p），加起來平均就是 r/p。

💡 與二項關係：P(X≤k) ↔ 累積成功次數，兩者互為「看次數」與「看成功數」的不同角度。
多項式分配 Multinomial（多項式分配）Week 2-28 · 補充
用在哪：二項分配的「多類別」版 — 每次試驗有 k 種結果（不只成功/失敗），機率 p₁…p_k。問各類別各出現幾次。

機率函數

f(x₁,…,xk) = (nx₁,…,xk) p₁x₁·p₂x₂···pkxk

條件：Σxᵢ = n、Σpᵢ = 1、p 固定、放回抽樣。

每類期望 μxᵢ = n·pᵢ
每類變異 σ²xᵢ = n·pᵢ(1−pᵢ)
