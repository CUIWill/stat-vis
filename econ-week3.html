PART 1  ·  從個體走向總體

上兩堂我們研究「一個人、一家廠商」；這堂我們把鏡頭拉遠，看「整個國家」的經濟。

複習：個體經濟的地圖p.2~13

個體經濟（Microeconomics）研究單一消費者與廠商如何在市場上互動，最後匯集成價格。總體經濟（Macroeconomics）則把所有人加總起來，研究整個經濟體的表現。

消費者行為

買方

無異曲線（消費偏好）、預算線（消費限制）、效用最大消費（消費決策）。

廠商行為

賣方

等產量線（生產組合）、等成本線（生產限制）、最小成本生產（生產決策）。

市場價格

供需交會

需求＋供給 → 市場均衡價格。完全競爭下廠商是價格接受者；非競爭市場廠商是價格制定者。

🔭 一句話分辨個體 vs 總體

個體（Micro）
一顆樹：某家公司的定價、某個人的消費選擇。
總體（Macro）
整片森林：全國的 GDP、失業率、通膨、經濟成長。
🔖 真實案例

台灣近年經濟成長率（GDP 年增率）像坐雲霄飛車

2021 Q1 疫後爆發 +8.16% → 2023 Q1 出口衰退 −3.02%（金融海嘯以來最差）→ 2024 Q1 反彈 +6.51%（11 年新高）→ 受 AI 與半導體帶動，2025 全年上修至 8.63%，2026 Q1 更衝上 14.55%（近 48 年新高）。這些新聞標題背後的主角，就是本堂的核心：GDP 與經濟成長。

🧭 讀新聞的方法：新聞說的「經濟成長率」＝實質 GDP 的年增率。數字為正代表比去年生產更多、經濟在擴張；為負代表衰退。
何謂總體經濟p.11~13

定義

總體經濟學（Macroeconomics）研究整個經濟體系的整體（總合）績效。核心是衡量財貨與勞務的總產出，也就是實質國內生產毛額（GDP, Gross Domestic Product）。

消費 C

家戶購買的財貨與勞務。

毛投資 I

私部門新購買的資本財——設備和建築物。

政府購買 G

政府對財貨與勞務的購買。

淨輸出 X−M

出口減進口的淨額。

總合就業

Employment

在職人數——有工作的人。

總合失業

Unemployment

沒有工作而且正在找工作的人數。（沒在找工作的不算失業。）

PART 2  ·  GDP 國內生產毛額

衡量一個國家「一年生產了多少」的總指標，怎麼定義、怎麼算、有什麼盲點？

GDP 是什麼p.14

國內生產毛額（GDP, Gross Domestic Product）

一定時期內（一季或一年），一個區域經濟活動中所生產出的全部最終成果（產品和勞務）的市場價值（market value）。它是國民經濟核算的核心指標，用來衡量一國經濟狀況與發展水準。

🏭 國內

只算「在這塊土地上」生產的，不管生產者是本國人還是外國人。

🎯 最終成果

只算最終產品，避免重複計算。麵粉做成麵包，只算麵包，不重複算麵粉。

💰 市場價值

用市場價格加總。沒有市場交易的（如自己煮飯）不計入。

💡 記憶訣竅：GDP＝「這一年，這塊土地上，生產出來能賣錢的東西，全部加起來值多少錢」。
名目 GDP vs 實質 GDPp.15~17
名目 GDP

Nominal GDP

用當年價格衡量一段期間內生產出來的財貨總值。物價漲，名目 GDP 就跟著虛胖。

實質 GDP

Real GDP

名目 GDP 除以隱含物價水準；或用基期價格計算該年份的總產值，剔除通膨，反映真實產量。

📐 實質 GDP 成長率（經濟成長率）

成長率 = Yt − Yt−1 Yt−1 × 100%

其中 Yt 是今年的實質 GDP，Yt−1 是去年的實質 GDP。乘以 100 就得到百分率表示的成長率。這正是新聞裡「經濟成長率 8.16%」的算法。

🧭 讀圖方法：新聞的成長率折線圖（yoy＝與去年同期相比），線在 0 以上代表成長，跌破 0 代表衰退（如 2023 Q1 的 −3.02%）。
⚠️ 常見陷阱：名目 GDP 變大，不代表大家真的變富有——可能只是物價漲了。比較不同年份的經濟表現，一定要看實質 GDP。
GDP 的三種衡量法p.18~21

同一塊大餅可以從三個角度算，結果理論上相等：生產它 → 分配所得 → 花掉它。

① 所得面

誰賺走了

用各生產要素賺取的所得加總，稱為國民所得（national income）：薪資、租金、利潤、利息等。

② 生產面

哪個產業做的

把國民所得依各生產部門細分：製造業、金融保險、批發零售、服務業⋯⋯

③ 支出面

誰買走了

從最終購買者的支出估計，最常用。就是下面的 C+I+G+(X−M)。

📐 支出面公式（最常用）

GDP = C + I + G + (X − M)
C　私人消費開支
I　投資總額
G　政府消費開支
X　出口毛額
M　進口毛額
💡 為什麼要「減進口」？因為 C、I、G 裡面混進了買外國貨的錢，那不是「國內」生產的，要扣掉。
GDP 的缺點與重要性p.22~24
缺點：它沒算到的東西
  • 未考慮所得分配的變動（貧富差距看不出來）
  • 排除了非市場財貨（自己帶小孩、做家事沒算）
  • 未給予休閒時間任何價值
  • 未考慮環境損害，如空氣和水的品質
重要性：為何仍是核心指標

儘管有盲點，實質 GDP 仍是衡量一國生活水準與景氣的最佳單一指標——它可跨年、跨國比較，資料齊全且更新快，是政策制定、投資判斷的共同語言。

⚠️ 考試提示：「實質 GDP 的四大缺點」很常考——所得分配、非市場財貨、休閒、環境。用「分、非、休、環」四個字記。

PART 3  ·  經濟成長

國家怎麼「一年比一年生產更多」？供給面（生產面）的視角：Solow 模型。

經濟成長與生產函數p.29~35

什麼是經濟成長（economic growth）

指實質國民所得或產出，隨時間經過而增加的現象。畫在生產可能線（PPF）上，就是整條線往外推。

生產函數——古典學派

Y = F(K, L)

兩種要素投入：資本存量 K（capital stock，機器廠房）與勞動 L（labor，工人）。古典學派認為：勞動力增加＋資本累積，就創造了經濟成長。

生產可能線（PPF）外推＝經濟成長
🧭 讀圖：線離原點越遠＝能生產越多。往外推＝經濟成長，往內縮＝衰退。
Solow 模型與技術進步p.34~40

生產函數——新古典學派（Solow 模型）

Y = A · F(K, L)

在資本 K、勞動 L 之外，再加上第三個要素——技術進步 A（advanced technology）。觀察式子：Y 和 A 呈比例變動，若 A 倍增而 K、L 不變，Y 也倍增。這解決了「光靠增加人和機器，成長會停滯」的困境。

📐 Solow 成長分解式

ΔYY = ΔAA + α·ΔKK + β·ΔLL

白話：經濟成長率（ΔY/Y）＝ 技術成長（ΔA/A）＋ 資本貢獻（α·ΔK/K）＋ 勞動貢獻（β·ΔL/L）。技術進步、資本、勞動，就是生產函數的三項要素。

🚀 技術進步：四次工業革命

① 第一次（1760–）蒸汽機
② 第二次（1870–）電力
③ 第三次（1945–）電腦/資訊
④ 第四次（2010–）AI/物聯網
技術進步 A↑ → 生產函數整條上移
🧭 讀圖：曲線凹向下＝資本報酬遞減（機器越加越多，多出來的產量越少）。技術 A 進步讓整條線往上跳。
💡 記憶訣竅：古典學派「K + L」兩要素 → Solow 加一個「A」變三要素。台灣近年 GDP 飆高，主角就是 A：AI 與半導體技術帶動。
均衡時的消費、儲蓄與投資p.44~45

關鍵方程式（家戶預算限制的總合形式）

C + ΔK = Y − δK

左邊：消費 C ＋ 淨投資 ΔK；右邊：實質 GDP（Y）− 折舊（δK）＝ 實質國內生產淨額。意思是：一國生產出來的東西扣掉機器耗損後，不是被消費掉，就是拿去累積成新資本（投資）。家戶消費 C 選得越少，留給投資 ΔK 的就越多。

ΔK　淨投資（新增資本）
δ　折舊率（機器耗損）
δK　折舊金額
💡 這條式子是連結「成長理論」與下一節「凱因斯模型」的橋梁：儲蓄（沒消費的部分）＝可用來投資的資金。

PART 4  ·  凱因斯模型

需求面（支出面）的視角：是「花錢（需求）」帶動了生產，還是「生產（供給）」創造了需求？

古典學派 vs 凱因斯學派p.42~43

1930 年代大蕭條是分水嶺。之前古典學派當道，之後凱因斯學派興起。兩派對「價格會不會自己調整」「政府該不該出手」看法完全相反。

古典學派（1930 前）
  • 價格（工資）有伸縮性（會漲也會跌）
  • 充分就業是常態
  • 供給創造本身需求（賽伊法則）
  • 重視貨幣政策
  • 反對政府干預
凱因斯學派（1930 大蕭條後）
  • 價格（工資）有僵硬性（不易下跌）
  • 失業是常態
  • 需求才能創造供給
  • 重視財政政策
  • 政府要干預經濟活動
💡 一句話對比：古典說「東西做出來自然賣得掉」；凱因斯說「大家不敢花錢，東西就賣不掉，政府得帶頭花」。景氣差時政府發振興券、普發現金，就是凱因斯的思路。
🔖 真實案例

政府「帶頭花錢」刺激需求

台灣：2015 年 GDP 保 1 → 政院砸 40.8 億刺激消費；2020 振興三倍券、2021 五倍券、2023 普發現金 6000（編列 1416.5 億）、2025 普發一萬。前瞻基礎建設 8800 億帶動 GDP。國外：美國拜登 1.2 兆美元基建法案、沙烏地 1 兆美元蓋未來城、韓國青瓦台砸 16.5 兆蓋「韓版南科」。這些都是用 G（政府支出）與 I（投資）拉抬總需求。

消費函數與儲蓄函數p.51~55

總需求（總合支出）的四元素

經濟體對產出的需求總量稱為總合支出（總合需求），由四部分組成：消費 C、投資 I、政府支出 G、淨出口 NX（＝出口 X − 進口 M）。

消費函數
C = a + c(Y − T) = a + c·Yd
a：自發性消費（autonomous consumption）——與所得無關、生存也得花的錢。
c：邊際消費傾向（MPC, marginal propensity to consume）＝ΔC/ΔYd，介於 0 與 1。多賺 1 元會花掉的比例。
Yd：可支配所得（＝所得 Y − 稅 T）。
儲蓄函數
S = −a + (1−c)Yd = −a + s·Yd
沒被消費掉的所得就是儲蓄：S = Yd − C。
−a：自發性儲蓄（負儲蓄）——所得為 0 時得動老本。
s：邊際儲蓄傾向（MPS）＝1−c。多賺 1 元會存起來的比例。
必然關係：c + s = 1（賺的錢不是花就是存）。

💰 利率對消費與儲蓄的效果（圖 7-4）

跨期替代效果：利率↑ → 存錢更划算 → 減少消費、增加儲蓄。
所得效果：利率↑ → 利息收入變多、覺得變有錢 → 增加消費、減少儲蓄。
⚠️ 兩效果方向相反，實際淨效果要看誰大。這也是為何有些國家會用「負利率」逼大家別存錢、多花錢（如歐洲央行、日本）。
投資函數與均衡所得p.57~61

投資支出函數

預擬投資 Ip（planned）：廠商「想達成」的投資量。
實際投資 Ir（realized）：實際達成的投資量。
當 Ip = Ir 時廠商不再調整生產，此時所得＝均衡所得。
簡化假設：投資為常數（自發性投資）I = Ia。

📐 均衡所得（無政府部門）

均衡時總產出 Y ＝ 總需求 Yd。解出：

Y* = a + Ia 1 − c

白話：均衡所得＝（自發性消費＋自發性投資）÷ 邊際儲蓄傾向（1−c）。分母 1−c 越小（越愛花錢），所得被放大越多，這就是乘數效果。

凱因斯 45° 線：總需求＝總產出
🧭 讀圖：藍線（總需求）與黑色 45° 線的交點 E，就是均衡。E 對應的橫軸位置 Y* 即均衡所得。45° 線上任一點都滿足「需求＝產出」。

另一種求法：投資＝儲蓄（I = S）

均衡時 Yd = C + I = Y = C + S，兩邊消去 C 得 I = S。把儲蓄函數 S = −a + (1−c)Y 代入 I = Ia，同樣解出 Y* = (a + Ia)/(1−c)。兩種方法答案一致。

PART 5  ·  儲蓄的兩面

「儲蓄愈多愈好」嗎？短期會拖垮景氣（節儉悖論），長期卻能促進成長。

節儉的矛盾（節儉的悖論）p.62~66

什麼是節儉的矛盾（Paradox of Thrift）

當大家自發性儲蓄增加（儲蓄線 S₁ 上移到 S₂），均衡所得會下降（Y₁ → Y₂），而總儲蓄量卻沒有增加。想省錢反而讓整體變窮——這就是矛盾。

📐 自發性支出乘數

ΔYΔa = 11 − c

自發性儲蓄增加 1 單位（即少花 1 元），均衡所得減少 1/(1−c) 單位。所得減少後，誘發性儲蓄剛好減少 1 單位，抵銷掉原本多存的 1 單位 → 總儲蓄不變。

儲蓄線上移 → 所得下降
🧭 讀圖：藍色↑箭頭＝大家更愛儲蓄，儲蓄線左上移；紅色←箭頭＝均衡所得從 Y₁ 縮到 Y₂。與投資線 I 的交點決定均衡。
⚠️ 考試提示：節儉悖論成立的前提是「短期、封閉經濟、需求不足」。個人省錢是美德，但全體同時省錢，會讓需求崩、所得縮。這正是凱因斯反對「大蕭條時緊縮」的理由。
儲蓄的好處（長期視角）p.67

① 促進資本累積

存下來的錢化為投資，累積機器廠房（K↑）。

② 融通投資

一國要提高投資，須降低消費、提高儲蓄。整體儲蓄＝私人儲蓄＋政府儲蓄。

③ 可貸資金供給增加

整體儲蓄提高 → 可貸資金（SLF）供給增加。

💡 短期 vs 長期不矛盾：需求不足時（短期）別急著存；經濟正常時（長期）儲蓄才能養出資本、支撐成長。這也呼應 Solow 模型的 K。
可貸資金市場（Loanable Funds）
🧭 讀圖：紅色 SLF（儲蓄＝資金供給）向右上、藍色 DLF（投資＝資金需求）向右下，交點決定均衡利率 r₀。儲蓄增加會使 SLF 右移，利率下降、投資增加。

PART 6  ·  政府部門

把政府（G 與稅 T）放進凱因斯模型，均衡所得公式怎麼變？

政府支出與預算限制p.73~75

政府支出的定義

各級政府購買財貨與勞務、移轉支付（發給家戶與企業的錢，如補助）及利息支付的總金額。

政府的預算限制（Governmental Budget Constraint）

Gt + Vt = Tt + (Mt − Mt−1) / Pt

白話：資金使用總額 ＝ 資金來源總額。
左邊（用途）：實質購買 G ＋ 實質移轉 V。
右邊（來源）：實質稅收 T ＋ 貨幣創造的實質收入（印鈔票）。

公共生產

政府用 G 提供服務給家戶與企業，且免費提供：國防、警消、義務教育、部分健康服務、高速公路、公園等。

加入政府部門的均衡所得p.76~77

模型設定

總需求：Yd = C + I + G。
消費函數：C = a + c(Y − T)。
假設投資 Ia、政府支出 Ga、稅收 Ta 均為常數。
均衡時 Y = Yd：
Y = a + c(Y − Ta) + Ia + Ga

📐 均衡所得（含政府）

Y* = a − c·Ta + Ia + Ga 1 − c

政府支出 G↑ 或減稅 T↓，都會讓分子變大 → 均衡所得放大。這就是財政政策刺激景氣的原理。

含政府的 45° 線
🧭 讀圖：加入 G 後，總需求線（藍）整條往上抬，與 45° 線的交點 E 右移 → 均衡所得 Y* 變大。

PART 7  ·  開放經濟與國際貿易

再把「出口 X 與進口 M」放進來——台灣是出口導向國家，這一段特別關鍵。

加入國外部門與貿易餘額p.82~86

開放經濟的總需求

Yd = C + I + G + (X − M)

假設出口為常數 X = Xa；進口函數 M = b + mY。
b：自發性進口（與所得無關）。
mY：誘發性進口（隨所得增加）。
m：邊際進口傾向（MPI），0 < m < 1，多賺 1 元會拿去買外國貨的比例。

📐 均衡所得（含政府＋開放）

Y* = a − b − c·Ta + Ia + Ga + Xa 1 − c + m

分母多了 +m：進口是「漏出」（錢流到國外），會削弱乘數效果。

貿易餘額（Trade Balance）＝淨輸出

貿易餘額 = Yt − (Ct + It + Gt)
= 實質 GDP − 實質國內支出 = 出口 − 進口
出口 > 進口
貿易餘額為正 → 盈餘
進口 > 出口
貿易餘額為負 → 赤字
🔖 真實案例

台灣是出超（貿易盈餘）國家，AI 與半導體出口強勁是近年 GDP 飆升的主因（淨輸出 X−M 大幅為正）。這也是為何「出口不妙」時（2023 Q1）GDP 會轉負。

貿易條件與投資p.84~89

貿易條件（Terms of Trade）

本國商品價格 P 相對外國商品價格 Pf 的比值 P / Pf。意義：每出口 1 單位本國商品，能換回多少單位外國商品。

P/Pf 上升（貿易條件改善）
每單位本國貨換到更多外國貨 → 本國福利↑
P/Pf 下滑（貿易條件惡化）
換到的外國貨變少 → 本國福利↓

貿易條件與投資的連動

資本的實質淨報酬率 = (P/Pf)·MPK − δ。當 P/Pf 提高，效果類似技術水準 A 提高：MPK 上升 → 資本報酬率上升 → 國內淨投資（I − δK）反應很大，通常大過全國儲蓄的增加。

💡 記憶訣竅：貿易條件改善 ≈ 技術進步，都會讓「投資變得更划算」，帶動 K 累積與成長。
⚠️ 政府常用關稅（tariff，對進口財課稅）或配額（quota，限制進口數量）限制貿易，但這會犧牲生產效率——由各國依相對要素稟賦「集中生產」才最有效率。

PART 8  ·  促進成長的政策 & 總覽

政府能做什麼來拉高長期成長率？最後用一張圖串起整堂總體經濟。

促進經濟成長的政策p.90~95
鼓勵投資

投資租稅減免 → 投資增加 → 資金需求曲線右移、利率提高。（例：台版晶片法研發支出 25% 抵減營所稅）

促進技術進步

用相同資源生產更多或更好的產品，也就是 Solow 模型的 A。

政策制訂者促進技術進步的四種手段

  • 直接資助基礎研究：知識具公共財性質，基礎研究常由政府資助。
  • 租稅政策：透過減稅鼓勵廠商研發（R&D）。
  • 保護智慧財產權：給研發成果一定期限的保護（專利），才有誘因創新。
  • 促進競爭：打擊獨占、提高競爭的政策或法令。
🔖 真實案例

中美貿易戰促台商「鮭魚返鄉」、境外資金匯回專法（實際匯回 3,351 億、創造國內投資 1,757 億）、台版晶片法（研發 25% 抵減營所稅、鼓勵企業根留台灣）——都是政府用政策拉抬投資 I 與技術 A 的實例。

Solow 成長分解p.35

ΔY/Y = ΔA/A + α·(ΔK/K) + β·(ΔL/L)
